# 利用規約

最終更新日: 2024年12月18日

## 1. はじめに

本利用規約（以下「本規約」）は、「AI参加会議」（以下「本アプリ」）の利用条件を定めるものです。本アプリを利用することにより、ユーザーは本規約に同意したものとみなされます。

## 2. サービスの説明

本アプリは、会議中の音声をリアルタイムで文字起こしし、AIによる要約・意見・アイデア提案を行うWebアプリケーションです。

### 2.1 提供される機能

- 音声の文字起こし（Gemini Audio / OpenAI Whisper）
- AI による会議内容の要約
- AI による意見・分析の提供
- AI によるアイデア提案
- 会議記録のエクスポート

### 2.2 必要なもの

- 対応ブラウザ（Chrome / Edge 推奨）
- マイク（音声入力用）
- 各AIサービスのAPIキー（ユーザー自身で取得）

## 3. 利用条件

### 3.1 APIキーについて

本アプリを使用するには、以下のいずれかのAPIキーが必要です：

- Google Gemini API キー
- OpenAI API キー
- Anthropic Claude API キー
- Groq API キー

これらのAPIキーは、ユーザー自身が各サービスの利用規約に同意の上、取得する必要があります。

### 3.2 利用料金

本アプリ自体は無料で提供されますが、各AIサービスの利用には料金が発生する場合があります。API利用料金は、各サービス提供元に直接支払われます。本アプリに表示されるコストは概算であり、実際の請求額を保証するものではありません。

### 3.3 年齢制限

本アプリは13歳以上の方を対象としています。13歳未満の方は、保護者の同意と監督のもとでご利用ください。

# プライバシーポリシー

最終更新日: 2024年12月18日

## はじめに

「AI参加会議」（以下「本アプリ」）は、ユーザーのプライバシーを尊重し、個人情報の保護に努めます。本プライバシーポリシーは、本アプリがどのような情報を収集し、どのように使用するかを説明します。

## 収集する情報

### 本アプリが収集しない情報

本アプリは以下の情報を**収集しません**：

- 個人を特定できる情報（氏名、メールアドレス、電話番号など）
- 会議の音声データや文字起こし内容
- APIキー（外部サーバーへの送信なし）
- 使用状況やアクセスログ
- Cookie

### ローカルに保存される情報

以下の情報は、ユーザーのブラウザのローカルストレージにのみ保存されます：

| 情報 | 目的 | 暗号化 |
|------|------|--------|
| APIキー | 各AIサービスへのアクセス | ✅ 暗号化して保存 |
| モデル設定 | ユーザーの選択を記憶 | ❌ |
| オプション設定 | コストアラートなど | ❌ |
| 初回訪問フラグ | ウェルカム画面の表示制御 | ❌ |

これらの情報は：
- ユーザーのデバイス内にのみ存在します
- 外部サーバーには一切送信されません
- ブラウザの設定から削除できます
- 「設定」画面から一括削除できます

## 外部サービスへの通信

本アプリは、ユーザーが明示的に操作した場合にのみ、以下の外部サービスと通信します：

### AI APIサービス

| サービス | 用途 | 送信されるデータ |
|----------|------|------------------|
| Google Gemini API | 文字起こし、AI回答 | 音声データ、テキストプロンプト |
| OpenAI API | 文字起こし（Whisper）、AI回答 | 音声データ、テキストプロンプト |
| Anthropic Claude API | AI回答 | テキストプロンプト |
| Groq API | AI回答 | テキストプロンプト |

これらの通信は：
- すべてHTTPS（暗号化通信）で行われます
- ユーザーが「録音開始」や「要約」などのボタンを押した時のみ発生します
- 各サービスのプライバシーポリシーに従って処理されます

### 各サービスのプライバシーポリシー

- [Google プライバシーポリシー](https://policies.google.com/privacy)
- [OpenAI プライバシーポリシー](https://openai.com/policies/privacy-policy)
- [Anthropic プライバシーポリシー](https://www.anthropic.com/privacy)
- [Groq プライバシーポリシー](https://groq.com/privacy-policy/)

## データの保持

### ローカルデータ

- **セッション中のデータ**（文字起こし内容、AI回答）：ブラウザを閉じると消去されます
- **設定データ**（APIキー、オプション）：明示的に削除するまで保持されます

### 自動削除オプション

「ブラウザを閉じたらAPIキーを自動削除」オプションを有効にすると、ブラウザを完全に閉じた際にAPIキーが自動的に削除されます。

## データの安全性

本アプリは以下のセキュリティ対策を実施しています：

1. **暗号化**: APIキーはデバイス固有のキーで暗号化して保存
2. **ローカル完結**: データは外部サーバーに送信されません
3. **HTTPS通信**: API呼び出しはすべて暗号化通信
4. **Same-Origin Policy**: 他のWebサイトからのデータアクセスをブラウザが防止

## ユーザーの権利

ユーザーは以下の権利を有します：

- **アクセス権**: ローカルストレージに保存されたデータを確認できます
- **削除権**: 「設定」画面からすべてのデータを削除できます
- **エクスポート権**: 設定データを暗号化ファイルとしてエクスポートできます

## 子どものプライバシー

本アプリは13歳未満の子どもを対象としていません。13歳未満の方は、保護者の同意なく本アプリを使用しないでください。

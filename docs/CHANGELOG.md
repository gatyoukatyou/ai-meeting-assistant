# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.4.0] - 2024

### Added
- リアルタイム音声ビジュアライザー
- 複数のAIモデル選択（GPT-4o, GPT-4o mini, GPT-4 Turbo, GPT-3.5 Turbo）
- 自動要約機能（設定可能な間隔）
- アイデア生成機能
- 文字起こしデータの保存機能（テキストファイル）
- API Keyのローカルストレージ保存

### Changed
- UIデザインの刷新（ダークモード、グラデーション）
- レスポンシブレイアウトの改善

### Fixed
- 音声認識の自動再開機能
- 長時間録音時の安定性向上

## [0.3.0] - 2024

### Added
- AIによる意見・提案機能
- 設定パネル
- 録音状態インジケーター

### Changed
- パネルレイアウトの2カラム化

## [0.2.0] - 2024

### Added
- OpenAI API連携
- 会議要約機能
- エラーハンドリングの改善

### Fixed
- 文字起こしの精度向上

## [0.1.0] - 2024

### Added
- 初期リリース
- Web Speech APIによる音声認識
- 基本的な文字起こし機能
- シンプルなUI

# 09 採用進捗会議

## 目的
採用ファネルを可視化し、採用目標達成に向けたボトルネックを解消する。

## コピー用AIワークオーダー
```text
会議タイプ: 採用進捗会議
対象期間: [月次]
対象職種: [例: エンジニア, セールス]

以下の順で出力してください。
1. 採用ファネル（応募/書類通過/面接/オファー/承諾）
2. ボトルネック工程トップ3
3. 改善施策（短期で効く順）
4. 今月の採用計画への影響
5. 実行タスク（担当/期限/完了条件）
6. 不足情報
7. 上司向け3行報告
```

## 必須確認事項
- 職種別にデータが分かれているか
- ボトルネックの原因が仮説止まりか事実か明示されているか
- 施策の効果測定指標が設定されているか

## 上司向け3行報告フォーマット
- 事実: [ファネル実績]
- 課題: [詰まり工程]
- 対応: [直近2週間の改善施策]

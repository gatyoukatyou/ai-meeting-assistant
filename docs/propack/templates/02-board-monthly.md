# 02 取締役会（月次）

## 目的
取締役会向けに、意思決定に必要な論点を1ページで把握できる状態にする。

## コピー用AIワークオーダー
```text
会議タイプ: 取締役会（月次）
対象期間: [対象月]
対象: 取締役会

以下の順で1ページ相当の出力を作成してください。
1. エグゼクティブサマリー（3段落以内）
2. 取締役会への決裁依頼（Yes/No判断が必要な項目）
3. 財務インパクト（売上・利益・キャッシュ）
4. 主要リスクと回避策
5. 決裁後48時間の実行タスク（担当/期限/完了条件）
6. 不足情報
7. 上司向け3行報告
```

## 必須確認事項
- 決裁依頼項目が「判断可能な粒度」か
- リスクの影響度と発生確率が混同されていないか
- 財務インパクトが金額またはレンジで示されているか

## 上司向け3行報告フォーマット
- 事実: [決裁が必要な論点と金額]
- 課題: [意思決定を遅らせる要因]
- 対応: [決裁後48時間アクション]

# 01 経営会議（週次）

## 目的
週次で経営KPIの変動要因を確認し、今週中の意思決定と実行項目を確定する。

## コピー用AIワークオーダー
```text
会議タイプ: 経営会議（週次）
対象期間: [今週の日付範囲]
対象: 全社

この会議ログから以下の順で出力してください。
1. エグゼクティブサマリー（最大5行）
2. KPIダッシュボード（売上, 粗利率, burn rate, 採用充足率）
3. 先週からの差分（増減要因を3点）
4. 今週の意思決定事項（期限つき）
5. 次アクション（担当/期限/完了条件）
6. 不足情報
7. 上司向け3行報告
```

## 必須確認事項
- KPIの分母分子が会議内で一致しているか
- 意思決定者が未設定の項目が残っていないか
- 期限が今週内か来週以降かを明示しているか

## 上司向け3行報告フォーマット
- 事実: [数値で1行]
- 課題: [ボトルネックを1行]
- 対応: [いつまでに誰が何をするか1行]

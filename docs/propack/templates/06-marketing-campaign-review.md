# 06 マーケ施策レビュー

## 目的
実施中キャンペーンの投資対効果を整理し、次サイクルの配分判断を行う。

## コピー用AIワークオーダー
```text
会議タイプ: マーケ施策レビュー
対象期間: [キャンペーン期間]
対象チャネル: [広告/SNS/イベント]

以下の順で出力してください。
1. 施策別サマリー（費用/成果/KPI）
2. 期待比で良かった点・悪かった点
3. 継続/停止/改善の判断案
4. 次サイクル配分案（%または金額）
5. 実行タスク（担当/期限/完了条件）
6. 不足情報
7. 上司向け3行報告
```

## 必須確認事項
- CPA/CACなど指標定義が会議内で一致しているか
- 施策評価が短期指標だけに偏っていないか
- 配分案に「減らす施策」と「増やす施策」が両方あるか

## 上司向け3行報告フォーマット
- 事実: [最も効いた施策と指標]
- 課題: [ROIが悪い施策]
- 対応: [次サイクル配分の変更]

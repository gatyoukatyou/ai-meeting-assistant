# 11 ベンダー折衝会議

## 目的
契約条件・価格・納期の交渉論点を整理し、合意形成までの打ち手を明確化する。

## コピー用AIワークオーダー
```text
会議タイプ: ベンダー折衝会議
対象ベンダー: [社名]
対象契約: [契約名]

以下の順で出力してください。
1. 交渉論点一覧（価格/納期/品質/責任分界）
2. 相手主張と自社主張
3. 合意済み項目と未合意項目
4. 次回交渉に向けた戦略案
5. 実行タスク（担当/期限/完了条件）
6. 不足情報
7. 上司向け3行報告
```

## 必須確認事項
- 合意済みと未合意が混在していないか
- 数値条件（単価、SLA、違約条件）が明記されているか
- 法務確認が必要な項目が抽出されているか

## 上司向け3行報告フォーマット
- 事実: [交渉の現在地]
- 課題: [未合意の最大論点]
- 対応: [次回交渉までの準備]

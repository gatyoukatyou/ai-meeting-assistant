# 12 予実差異レビュー（財務）

## 目的
予算と実績の差異を分解し、業績見通しと是正アクションを確定する。

## コピー用AIワークオーダー
```text
会議タイプ: 予実差異レビュー（財務）
対象期間: [対象月/四半期]
対象部門: [部門名]

以下の順で出力してください。
1. 予実差異サマリー（売上/原価/販管費/利益）
2. 差異要因分解（数量/単価/ミックス）
3. 通期見通しへの影響
4. 是正アクション（短期/中期）
5. 実行タスク（担当/期限/完了条件）
6. 不足情報
7. 上司向け3行報告
```

## 必須確認事項
- 差異が金額と率の両方で示されているか
- 一過性要因と継続要因が分離されているか
- 通期見通しに反映する前提が明示されているか

## 上司向け3行報告フォーマット
- 事実: [主要差異金額]
- 課題: [差異の主因]
- 対応: [是正アクションと締切]

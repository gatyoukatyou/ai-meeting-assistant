# 04 プロジェクト進捗会議（SteerCo）

## 目的
プロジェクトの進捗、遅延リスク、意思決定依頼を経営層が即判断できる形にする。

## コピー用AIワークオーダー
```text
会議タイプ: プロジェクト進捗会議（SteerCo）
対象プロジェクト: [プロジェクト名]
対象期間: [週次/月次]

以下の順で出力してください。
1. 全体進捗（計画比%）
2. マイルストーン達成状況（赤黄緑）
3. 重大課題（解決期限つき）
4. エスカレーション事項（意思決定者つき）
5. 次アクション（担当/期限/完了条件）
6. 不足情報
7. 上司向け3行報告
```

## 必須確認事項
- 遅延の原因と影響範囲が分かれているか
- エスカレーション先が実名または役職で特定されているか
- 次アクションがマイルストーンに紐付いているか

## 上司向け3行報告フォーマット
- 事実: [進捗率と遅延有無]
- 課題: [今週の最大リスク]
- 対応: [意思決定依頼と期限]

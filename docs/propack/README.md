# Pro Pack Templates v1

会議タイプ別テンプレート集です。`AIワークオーダー` に貼るだけで、会議後の出力フォーマットを固定できます。

## 使い方

1. 会議タイプに近いテンプレを選ぶ。
2. テンプレ内の「コピー用AIワークオーダー」をエクスポート時の `### AI Work Order` に貼る。
3. プレースホルダー（期間、対象プロジェクト、KPIなど）だけ埋める。

## 共通雛形

- [AI_WORK_ORDER_BASE.md](AI_WORK_ORDER_BASE.md)

## v1で統一している出力方針

- 事実と解釈を分離して出力する
- 決定事項と未決事項を分離する
- アクションは `担当 / 期限 / 完了条件` の3点を必須化する
- 末尾に「上司向け3行報告」を必ず付与する

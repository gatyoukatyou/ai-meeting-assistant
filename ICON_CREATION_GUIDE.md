# アイコン画像作成ガイド

PWAのホーム画面アイコンを作成するためのガイドです。

## 必要なアイコンサイズ

以下の2つのサイズが必要です：
- **192×192 px** → `icon-192.png`
- **512×512 px** → `icon-512.png`

## デザインガイドライン

### **テーマカラー**
- プライマリカラー: `#4f46e5`（紫）
- 背景色: `#f9fafb`（薄いグレー）

### **推奨デザイン**

#### **オプション1: シンプルなアイコン**
```
背景: #4f46e5（紫）
アイコン: 🤖（ロボット絵文字）+ 🎤（マイク絵文字）
テキスト: なし
```

#### **オプション2: テキスト入りアイコン**
```
背景: #4f46e5（紫）
テキスト: 「AI会議」（白色、太字）
アイコン: 🤖（中央上部）
```

#### **オプション3: モダンなアイコン**
```
背景: グラデーション（#4f46e5 → #6366f1）
図形: マイクの形状（シンプルな線画）
下部: 音波のような波線
```

## 作成方法

### **方法1: Canva（推奨・無料）**

1. [Canva](https://www.canva.com/)にアクセス
2. 「カスタムサイズ」で 512×512 px を作成
3. 背景色を `#4f46e5` に設定
4. 絵文字やテキストを追加
5. PNG形式でダウンロード → `icon-512.png`
6. 同じデザインで 192×192 px を作成 → `icon-192.png`

### **方法2: Figma（デザイナー向け・無料）**

1. [Figma](https://www.figma.com/)で新規ファイル作成
2. フレームを 512×512 px で作成
3. デザインを作成
4. Export → PNG → 512×512 px
5. 別フレームで 192×192 px を作成してエクスポート

### **方法3: AI画像生成（DALL-E, Midjourney など）**

プロンプト例：
```
Create a simple app icon for an AI meeting assistant.
- Background: solid purple (#4f46e5)
- Icon: modern microphone and robot symbol
- Style: minimalist, flat design
- Size: 512x512 pixels
- Format: PNG
```

### **方法4: オンラインツール（最も簡単）**

- [Favicon Generator](https://realfavicongenerator.net/)
- [PWA Icon Generator](https://www.pwabuilder.com/imageGenerator)

512×512 pxの画像を1つ作成すれば、自動的に192×192 pxも生成してくれます。

## ファイル配置

作成したアイコンを以下の場所に配置：

```
ai-meeting-assistant/
├── icon-192.png  ← ここに配置
├── icon-512.png  ← ここに配置
├── index.html
├── manifest.json
└── ...
```

## 確認方法

1. アイコンを配置後、GitHubにプッシュ
2. スマホでサイトにアクセス
3. 「ホーム画面に追加」を実行
4. ホーム画面にアイコンが表示されるか確認

## クイックスタート（最速）

時間がない場合は、以下の方法で仮アイコンを作成できます：

### **絵文字アイコン（1分で完成）**

1. 以下のサイトにアクセス：
   https://emoji-favicon.dev/

2. 絵文字「🤖」を選択

3. 背景色を `#4f46e5` に設定

4. 512×512 pxでダウンロード → `icon-512.png`

5. 192×192 pxでダウンロード → `icon-192.png`

---

**注意**: 実際のアプリとして公開する前に、プロフェッショナルなアイコンに差し替えることを推奨します。
